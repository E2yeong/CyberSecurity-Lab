1) What is a Network?

A network is a communication system that connects two or more devices to exchange information. In simple terms, it‚Äôs ‚Äúthe road that lets computers talk to each other.‚Äù

  • Smartphones, laptops, servers, routers‚Äîall operate over networks.
  • Networks can be visualized as graphs with nodes (devices) and links (connections).

7) Network Security ‚Äî CIA Triad

Network security ensures data remains confidential, unaltered, and available during transmission.

Confidentiality

Only authorized users access data.

Encryption, Access Control

Integrity

Data remains unchanged during transmission.

Hashes, Digital Signatures

Availability

Authorized users can access services when needed.

DDoS protection, Backups

8) Common Network Threats

  • Sniffing: capturing packets secretly (password theft).
  • MITM (Man-in-the-Middle): attacker intercepts communication (no HTTPS).
  • DDoS: overwhelm server with traffic.
  • Phishing: fake websites/emails.
  • Malware: system infection, ransomware.
  • Session Hijacking: stealing login sessions.
  • Cookie Theft: capturing stored credentials.

9) Defense Technologies & Multi-Layer Protection

  • Encryption: TLS (HTTPS), VPN
  • Authentication & Access Control: OTP, Certificates, VLAN isolation
  • Firewall: controls permitted traffic (e.g., WAF)
  • IDS/IPS: detect and prevent intrusions
  • DNSSEC / IPsec / Zero Trust: secure DNS, encrypted routing, continuous verification
