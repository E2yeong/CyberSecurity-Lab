Introduction

Linux commands are executed in the terminal (also known as the shell or command line). Mastering a few essential commands will help you navigate, manage files, and interact with the operating system efficiently. Below are some of the most widely used commands for beginners and professionals alike.

Frequently Used Commands

1. ls ‚Äì List Files

Displays the contents of the current directory.

$ ls
Desktop  Documents  Downloads  file.txt

Options:

  • ls -l ‚Üí long listing format (permissions, owner, size, date)
  • ls -a ‚Üí show hidden files (those starting with .)
  • ls -lh ‚Üí human-readable sizes (KB, MB, GB)

2. cd ‚Äì Change Directory

Navigate between directories.

$ cd /home/user/Documents

Tips:

  • cd ~ ‚Üí go to home directory
  • cd .. ‚Üí move up one directory

3. rm ‚Äì Remove Files/Directories

Deletes files or directories.

$ rm file.txt
$ rm -rf folder_name
‚ö†Ô∏è Warning: rm permanently deletes files‚Äîthere is no recycle bin.

4. cp ‚Äì Copy Files

Copies files or directories.

$ cp file.txt /home/user/backup/

Options:

  • cp -r folder/ /home/user/backup/ ‚Üí copy directories recursively

5. mv ‚Äì Move/Rename Files

Moves or renames files and directories.

$ mv file.txt /home/user/Documents/
$ mv oldname.txt newname.txt

6. cat ‚Äì View File Contents

Displays the content of a file.

$ cat file.txt

Useful with piping:

$ cat file.txt | grep "keyword"

Additional Useful Commands

  • pwd ‚Üí Print current working directory
  • touch ‚Üí Create an empty file
  • mkdir ‚Üí Create a new directory
  • rmdir ‚Üí Remove an empty directory
  • man ‚Üí Open manual pages for a command (e.g., man ls)
  • chmod ‚Üí Change file permissions
  • chown ‚Üí Change file ownership
  • grep ‚Üí Search inside files
  • find ‚Üí Search for files in directories
  • tar / gzip ‚Üí Archive and compress files

Conclusion

These commands form the foundation of Linux system interaction. Mastering them will allow you to navigate, manage files, and perform administrative tasks effectively. As you progress, you‚Äôll combine them with pipes (|) and redirection (>, <) to build powerful workflows.
